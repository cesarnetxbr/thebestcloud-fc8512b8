# Project Memory

## Core
Platform: The Best Cloud (B2B/B2C). Supabase backend with RLS. Modularity: always map existing code before expanding.
Design: Deep blue & vibrant orange. The Best Cloud brand only (no Acronis/Webplus references in public).
FinOps: TundraFinOps model. Invoices use COST- and SALE- prefixes. Focus on volume/security SKUs.
Tech: Supabase Edge Functions with Service Role Key for external APIs. Automated pg_cron syncs.
